# Fonia Labs Website

Fonia Labs is a global innovation group building companies, platforms, and products that solve real-world problems.

## Stack
- Next.js App Router
- React
- TypeScript
- Plain CSS

## Pages
- `/` Home
- `/companies`
- `/companies/[slug]`
- `/about`
- `/collaborate`
- `/updates`
- `/contact`

## Setup
```bash
npm install
npm run dev
```

## Deployment
Create a new GitHub repo named `fonia-labs`, push this project, then import it into Vercel.
